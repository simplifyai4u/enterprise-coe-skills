# Infrastructure as Code

Use this skill when automating environment provisioning and platform setup.

## Objectives
- Standardize infrastructure deployment
- Improve repeatability and change control
- Reduce manual configuration drift

## Instructions
1. Identify infrastructure components and lifecycle requirements.
2. Define configuration structure and reusable modules.
3. Recommend state management, secrets handling, and policy controls.
4. Summarize rollout and change management guidance.
