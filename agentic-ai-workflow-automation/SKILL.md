# Agentic AI and Workflow Automation

Use this skill when creating assistants or workflows that can reason, delegate, and act across tools.

## Objectives
- Define agent roles, goals, and boundaries
- Connect agents to tools and knowledge sources
- Improve reliability and controllability of automation

## Instructions
1. Identify tasks that can be automated or delegated.
2. Define agent responsibilities, inputs, and outputs.
3. Recommend tool use, guardrails, and fallback behavior.
4. Summarize validation and monitoring approach.
