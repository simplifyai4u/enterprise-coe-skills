# AI/ML Application Engineering

Use this skill when adding intelligent features or models to software systems.

## Objectives
- Define use cases and solution boundaries
- Recommend model and integration patterns
- Address quality, cost, and governance concerns

## Instructions
1. Clarify business problem and success criteria.
2. Select model approach and integration pattern.
3. Identify data needs, evaluation measures, and risk factors.
4. Summarize deployment and monitoring guidance.
