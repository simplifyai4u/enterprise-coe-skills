# Cloud Platform Engineering

Use this skill when you need to shape cloud architecture, platform choices, and operating models.

## Objectives
- Assess workload needs and target cloud services
- Recommend platform patterns and governance
- Define resilience, cost, and security considerations

## Instructions
1. Identify service, data, and compliance requirements.
2. Select appropriate cloud services and deployment patterns.
3. Outline platform capabilities, guardrails, and operating procedures.
4. Summarize trade-offs and recommended next steps.
