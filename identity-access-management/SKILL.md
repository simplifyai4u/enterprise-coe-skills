# Identity and Access Management

Use this skill when designing authentication, authorization, and access control approaches.

## Objectives
- Define user and service identity models
- Recommend access control and federation patterns
- Improve security and auditability

## Instructions
1. Identify actors, roles, and access needs.
2. Recommend authentication and authorization patterns.
3. Define secrets, tokens, and policy management requirements.
4. Summarize governance and audit guidelines.
