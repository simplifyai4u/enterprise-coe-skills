# Containerization and Kubernetes

Use this skill when deploying workloads in containers or Kubernetes platforms.

## Objectives
- Package services for portability and consistency
- Design Kubernetes deployment and scaling strategies
- Improve reliability and operational manageability

## Instructions
1. Review application packaging and runtime requirements.
2. Define container image strategy and dependency handling.
3. Outline deployment manifests, autoscaling, and resource controls.
4. Recommend troubleshooting and rollback practices.
