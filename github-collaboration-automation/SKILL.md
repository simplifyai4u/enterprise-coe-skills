# GitHub Collaboration and Automation

Use this skill when setting up repository workflows, collaboration practices, and automation.

## Objectives
- Improve pull request and review quality
- Standardize workflows and repository governance
- Automate recurring engineering tasks

## Instructions
1. Review current repository and workflow practices.
2. Define branching, review, and merge expectations.
3. Recommend Actions, templates, and automation patterns.
4. Summarize adoption and governance guidance.
