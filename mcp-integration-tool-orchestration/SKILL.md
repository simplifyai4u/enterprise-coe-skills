# MCP Integration and Tool Orchestration

Use this skill when building agent systems that need structured access to tools and services.

## Objectives
- Define tool interfaces and usage patterns
- Connect agents to capabilities and workflows
- Improve composability and maintainability

## Instructions
1. Identify tools, capabilities, and execution needs.
2. Define MCP-style contracts and interactions.
3. Recommend orchestration flow, error handling, and security controls.
4. Summarize implementation and governance guidance.
